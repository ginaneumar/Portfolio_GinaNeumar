Time Management System

Track your time, maximize your productivity!

Overview

Background

The task of the project is to design and implement a time management software.

Goal

I set myself the goal of designing a software to manage the worktime. That you only have to work normally with 2 Buttons "Start" and "Stop".

Ideation

Sketches

I want to create an intuitive design as software on the work computers. You work most of the time with a "Stop" and "Start" button for working time and then you can use a calendar for vacation, sickness.

Wireframe

Low-fidelity Wireframe

For the wireframe I followed the sketch and put the navgiation with the blue elements on the left side. At the top you can see the name and the remaining vacation days and the current working time. As a supervisor you have an extra button "Employees interact", so that the supervisor can approve a request.

Prototype

High-fidelity Prototype

In the high fidelity prototype you can already see how the Software can actually look like. We have transferred the blue elements from the sketch and wireframe also in the design and logo

The first start of the software begins with a login screen. There is also the possibility to report an error.

For working time and break information, we have adopted a simple design like standard calendars, so that everyone can work with it.

We discarded the idea of having a single "Employee Interact" button and instead added 2 buttons to add and remove employees and still access the information from the employees.

Reflection

What did I learn

It was the first time working in AdobeXD to design a software. I also worked with UML diagrams for the first time to get some insight behind the implementation.
